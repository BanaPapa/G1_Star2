## 1. 설계 원칙

```text
무기 성장 = 피해량 증가가 아니라 전투 행동의 확장
```

| 무기 계열 | 전투 문법 |
| --- | --- |
| Laser | 선을 긋고, 관통하고, 굴절시켜 적을 맞히는 정밀 무기 |
| Ion | 적의 시스템을 망가뜨려 다음 턴의 행동 가치를 떨어뜨리는 전자전 무기 |
| Plasma | 장갑과 전투 지속력을 녹이고, 폭발과 잔열로 전장을 압박하는 고열 무기 |
| Gravity | 적의 위치, 이동, 행동 비용을 조작하는 전장 제어 무기 |
| Antimatter | 이동계, 방어층, 무기, 공간, 존재 자체를 지우는 소멸 무기 |

한 줄 요약:

```text
Laser = 선을 긋는다
Ion = 시스템을 흔든다
Plasma = 장갑을 녹인다
Gravity = 위치를 비튼다
Antimatter = 존재를 지운다
```

## 2. Laser 계열

```text
Laser = 선형 조준 → 대각선 조준 → 관통 → 굴절 → 고급 관통 굴절
```

| 단계 | ID | 이름 | 핵심 특성 | 효과 요약 |
| -: | --- | --- | --- | --- |
| 1 | `wpn_laser_line` | Line Laser / 선형 레이저 | 직선 단일 타격 | 상하좌우 직선만. 처음 닿은 적 1기 피해 |
| 2 | `wpn_laser_vector` | Vector Laser / 벡터 레이저 | 대각선 조준 | 8방향 공격. 첫 번째 적 1기 피해 |
| 3 | `wpn_laser_pierce` | Piercing Beam / 관통 빔 | 1회 관통 | 같은 라인 두 번째 적까지 타격. 2번째는 50% 피해 |
| 4 | `wpn_laser_deflection` | Deflection Beam / 굴절 빔 | 기준점 굴절 | 장애물/아군/적군을 굴절점으로 1회 꺾임 |
| 5 | `wpn_laser_phase_lance` | Phase Lance / 위상 랜스 | 고급 굴절 관통 | 직선/굴절 경로 최대 3기. 1·2번째 100%, 3번째 50% |

### 판정 규칙
- 1단계는 상하좌우 4방향만, 2단계부터 8방향.
- 3단계 관통: 두 대상이 같은 직선/대각선 라인 위에 있어야 함. 중간 장애물은 차단.
- 4단계 굴절점: 장애물·아군 함선·적군 함선·특수 프리즘/왜곡 지형 가능. **빈 우주공간 불가.**
  - 적군을 굴절점으로 쓰면 그 적은 피격 대상이 됨. 아군/장애물 굴절점은 피해 없이 꺾이기만 함.
- 장애물은 기본적으로 레이저를 차단하나, 4단계부터 굴절점으로 사용 가능.

## 4. Plasma 계열

```text
Plasma = 아머 약화 → 공격력 약화 → 광역 폭발 → 강화 폭발 + 잔열 → 아머 붕괴 / 최대 HP 손상
```

| 단계 | ID | 이름 | 핵심 특성 | 효과 요약 |
| -: | --- | --- | --- | --- |
| 1 | `wpn_armor_melter` | Armor Melter / 아머 멜터 | 방어력 감소 | 1턴 방어력 -30%. 낮은 확률로 전투 종료까지 지속 |
| 2 | `wpn_core_melter` | Core Melter / 코어 멜터 | 방어/공격 감소 | 1턴 방어력·공격력 -30%. 낮은 확률로 전투 종료까지 |
| 3 | `wpn_plasma_burst` | Plasma Burst / 플라즈마 버스트 | 광역 폭발 | 5×5. 중심 100% / 1칸 80% / 2칸 60%. **아군도 피해** |
| 4 | `wpn_hellfire_burst` | Hellfire Burst / 헬파이어 버스트 | 강화 폭발 + 잔열 | 중심 120% / 1칸 100% / 2칸 80% + 잔열 지대 |
| 5 | `wpn_armor_annihilator` | Armor Annihilator / 아머 애니힐레이터 | 아머 붕괴 | 쉴드 무시 직접 피해. 일반 적 아머 무력화 / 최대 HP 손상 |

### 판정 규칙
- 4단계 잔열: 폭발과 동일한 5×5, 다음 턴까지 지속. 잔열 안에 남으면 최초 피해의 50% 도트 + 도트 받는 동안 장갑 방어력 50% 감소(벗어나면 해제). 즉시 폭발·잔열 모두 아군에게도 적용.
- 4단계 잔열의 장갑 감소는 1~2단계 멜팅(직접 약화)과 구분되는 **환경성 약화**.
- 5단계 일반 적: 쉴드 무시 직접 피해(쉴드 피해 없음), 아머 전투 종료까지 무력화, 높은 확률 최대 HP -30% / 낮은 확률 -50%.
- 5단계 보스: 아머 완전 무력화 없음. 높은 확률 방어도 -50%, 높은 확률 최대 HP -10% / 낮은 확률 -20%. 1회 적용 후 재중첩 불가.

### 중첩 규칙
| 효과 | 중첩 |
| --- | --- |
| 1단계 방어력 -30% | 불가 |
| 2단계 방어/공격 -30% | 불가 |
| 4단계 잔열 장갑 -50% | 잔열 안에 있을 때만 |
| 5단계 아머 무력화 / 보스 방어도 -50% | 1회만 |
| 5단계 최대 HP 감소 | 5단계 효과끼리 중첩 불가 |

- 4단계: 집결 후 중심점과 인접 1칸으로 흩어짐. 기존 중력장과 겹치면 수치 중첩 없이 지속 턴수만 추가.
- 5단계 보스: 이동 봉쇄 대신 이동력 50%, 공격력 70% 수준, 방어력 -30%.


| 2 | `wpn_defense_eraser` | Defense Eraser / 방어층 소거기 | 장갑/쉴드 소멸 | 높은 확률 장갑 내구도 -50% + 쉴드 -50%. 재피격 시 완전 붕괴 |
| 3 | `wpn_antimatter_field` | Antimatter Field / 반물질 에너지장 | 방어층 삭제 + 무기 폭주 | 높은 확률 장갑·쉴드 100% 영구 손실 + 2턴 피아 무차별 난사 |
| 4 | `wpn_micro_singularity` | Micro Singularity / 미소 특이점 | 블랙홀 생성 | 중심 주변 8칸 중 랜덤 4칸 블랙홀. 진입 시 소멸 |
| 5 | `wpn_total_annihilation` | Total Annihilation / 완전 소멸 병기 | 존재 소멸 | 직접 대상 높은 확률 완전 소멸. 인접 8칸 50% 연쇄 소멸 |

### 판정 규칙
- 1단계는 AP 전체가 아닌 **MOVE 태그 행동만 제한** (AP 태그: MOVE / ATTACK / GUARD / SKILL).
- 2단계: 이미 50% 절삭 상태에서 재피격 → 장갑 완전 파괴 + 쉴드 완전 붕괴(재충전 불가). 쉴드 등급 무관, 현재량 기준. 보스는 완전 파괴 불가 권장.
- 3단계 무기 폭주는 Ion 4단계 "피아식별 오류"와 구분: **무기 제어계 붕괴에 따른 난사**. 특수기 사용 불가 권장. 보스는 난사 대신 공격력 감소·쉴드 재충전 봉쇄 등 약화 적용.
- 4단계 블랙홀: 직접 피격 100% 피해 / 좌표 지정 시 중심 3×3에 50% 산탄. 중심점은 블랙홀이 되지 않음. 후보 8칸 중 랜덤 4칸, 전투 종료까지 지속. 이미 블랙홀인 칸이 선택돼도 보상 생성 없음. 진입 시 피아 구분 없이 소멸. 인접 1칸은 턴마다 무기/장갑/쉴드 내구도 -20%.
- 5단계: 소멸 실패 대상은 100% 쉴드 완전 파괴 + 70% 확률 무기·장갑 내구도 -50%(재피격 시 완전 파괴). **소멸은 처치 판정 — 경험치/아이템 정상 지급.** 보스 완전 소멸 불가.
- 보상 구분: Gravity 전장 이탈 = 보상 제외 / Antimatter 소멸 = 처치 판정, 보상 지급.

## 8. 구현 시 시스템 요구사항

### 스탯 분리 (Plasma vs Antimatter 구분 위해 필수)
```text
방어력 = 피해를 줄이는 수치
장갑 내구도 = 장갑 부품의 남은 상태
쉴드 충전율 = 현재 쉴드량
쉴드 재충전 = 턴마다 회복되는 쉴드량
```

### 지형 효과 타입
```text
잔열 지대 = 도트 피해 + 장갑 방어력 감소
중력장 = AP 비용 증가 + 공격력 감소 + 받는 피해 증가
블랙홀 = 진입 시 소멸 + 인접 장비 내구도 감소
```
(현재 코드 `terrain.js`에 residual_heat / gravity_well / black_hole 지형 존재 — Stage 3에서 필드효과로 구현됨)

### 보스 예외 규칙 (필수)
그대로 적용 금지: 전장 이탈, 이동 완전 봉쇄, 장갑/쉴드 완전 소멸, 무기 완전 파괴, 즉사/완전 소멸, 블랙홀 진입 즉사.

변환 규칙:
```text
즉사 → 최대 HP 비례 피해
이동 봉쇄 → 이동력 감소
전장 이탈 → 충격 피해 + 위치 이동 실패
완전 파괴 → 내구도 50% 손상
완전 소멸 → 쉴드 파괴 + 회복/재충전 봉쇄
```

## 9. 밸런스 의심 지점 (구현 시 관제실 조정 대상)

| 항목 | 위험성 | 대응 |
| --- | --- | --- |
| Gravity 3 받는 피해 2배 | 매우 강력 | 지속 짧게, 보스 약화 적용 |
| Gravity 4 + Plasma 4 연계 | 모아서 폭발 | 쿨다운/AP 비용 조정 |
| Gravity 5 이동 봉쇄 | 일반 적에 강력 | 보스는 이동력 감소로 |
| Antimatter 2 방어층 삭제 | 2단계치고 강력 | 완전 파괴는 낮은 확률/재피격 조건 |
| Antimatter 4 블랙홀 | 영구 즉사 지형 | 랜덤 4칸 제한, 보상 생성 없음 |
| Antimatter 5 연쇄 소멸 | 최종기지만 강력 | 보스/엘리트 즉사 금지 |
| Plasma 5 최대 HP 손상 | 장기전 영향 | 중첩 제한 |
| Ion 5 시스템 붕괴 | 복합 효과 | 전부 확정 금지, 일부 확률 |
