# 밸런스 전체 패스 — 난이도 곡선 분석 (Phase 7-5)

> **상태**: 분석 리포트 완료 (2026-07-08). **실수치 조정은 7-6 완주 플레이테스트 후 체감 기반으로 진행**한다
> (사용자 결정: "지금은 데이터 없이 config를 바꾸면 오히려 해로울 수 있으니 리포트만").
> 조정 시 관제실(System Control Room)에서 값을 만지고 → Export → `defaultGameConfig`/데이터에 반영하는 흐름.

## 1. 난이도 곡선 (정량)

성계 진행 순서 s1→s2→s3→s4. 적 스탯은 `encounter.js` threat 스케일(`m = 0.7 + (threat-1)×0.2`, EVA×0.3, DEF 미스케일) 반영.

| 성계 | threat | 스케일 | 적 수 | ΣHP | ΣATK | 보스 | 보상 |
|---|---|---|---|---|---|---|---|
| s1 아르카디아 | 2 | 0.90× | 2 | 108 | 54 | — | 카이 영입 + ti60 |
| s2 오르페우스 | 4 | 1.30× | 6 | 1141 | 299 | 가르(hp600) | 세라(조건) + dm8 + 배틀십 해금 |
| s3 크로노스 | 5 | 1.50× | 5 | 539 | 234 | — | 밀라 영입 |
| s4 니힐 | 7 | 1.90× | 7 | 2961 | 589 | 워든(hp1800) | 엔딩 |

**개별 유닛 스케일은 단조 증가(0.9→1.3→1.5→1.9×)로 정상.** 아래 특이점은 전부 *편성*(보스·적 수)에서 나온다.

## 2. 특이점 분석

### 2-1. s2 > s3 총 전투력 역전 (곡선상 골짜기)
- 진행상 뒤인 s3(ΣHP539)가 s2(ΣHP1141)의 절반. 원인은 **s2 가르 미니보스(hp600)** — 가르 제외 시 s2는 ΣHP541로 s3와 동급.
- **판단**: 의도된 페이싱일 가능성이 높다 — s2=미니보스 하이라이트, s3=숨고르기(밀라 영입·비밀상점 유령시장·다크매터 채굴). 개별 스케일은 정상이므로 "버그"는 아님.
- **만약 조정한다면**(플레이테스트 후): (a) s3 편성에 적 1기 추가해 진행순 단조 증가에 가깝게, 또는 (b) threatLevel 표기를 체감과 정렬(현재 s2=4 표기지만 총량은 s3 이상). 현행 유지도 방어 가능.

### 2-2. 가르 hp600 두께 (사용자 플래그)
- 4척(레이저 T1~2 장착, 총 유효딜 177/턴) 기준 **3.4턴 처치**(측면·크리 제외 최악치). 측면 +25%·크리 15%×1.8 감안 시 실측 ~2.5–3턴.
- 가르 atk55 → 초기 함급(건십 hp42·순양함 hp66)을 2방에 격파 → 3턴간 아군 1–2척 손실 위험. **초반 미니보스치고 hp600은 상단이지만 파탄은 아님.**
- **완화 장치 이미 존재**: 7-3에서 추가한 측후방 약점(포위 시 유효 DEF −50%) → 포위 기동하면 처치 턴이 크게 단축. 설계 의도(포위 보상)와 맞물림.
- **만약 조정한다면**: `enemies.json` 가르 `stats.hp` 600→500 소폭 하향, 또는 유지하고 약점 유도 튜토리얼/힌트 강화. **우선순위 낮음(약점으로 이미 커버)**.

### 2-3. s4 워든전 과부하 (사용자 플래그)
- 워든 hp1800 → 4척(T3 무기, 총딜 210/턴) 기준 **8.6턴**. 여기에 2페이즈 차원 파동(전체 30/턴) + 지휘함 증원 + void_rift 소환이 겹침.
- **판단**: "과부하"보다 **"장기전"** 성향. 8.6턴은 최종보스치고 긴 편 — 플레이어가 T4~T5 무기·완편 함대(6척)면 5–6턴으로 단축된다.
- **완화 장치 이미 존재**: 7-3 텔레그래프(파동 1턴 예고 → 흩어짐·방어로 카운터), 워든 약점 없음.
- **만약 조정한다면**: hp1800 유지가 최종보스 무게로는 타당. 장기전이 문제면 (a) T4~T5 무기 접근성(연구·유령시장·조합 7-4) 점검, (b) `combat.boss.warden.phase2AoeAtkMult`(현 0.5)·증원 빈도로 페이스 조절. **플레이테스트에서 "8턴이 지루한가/긴장되는가" 체감 판단 필수.**

### 2-4. s1→s2 진입 장벽 (사용자 플래그)
- threat 2→4 점프 + 적 2→6기 + 미니보스 첫 등장 = 첫 실질 관문의 급상승.
- s1은 적 2기(ΣHP108)로 매우 쉬움 → 카이 영입해 4척. 모항 상점 T1 무기·무기공학 I 연구로 대비 가능하면 넘을 만하나, **준비 없이 직행하면 벽**.
- **만약 조정한다면**: (a) s1에 적 1기 추가해 워밍업 강화, (b) s2 threat 4→3 소폭 하향(단, 배틀십 해금 보상과 균형), (c) s1↔s2 사이 경고/권장 전력 UI 힌트. **플레이테스트에서 "첫 s2 진입 승률" 체감으로 결정.**

## 3. 경제 곡선 (참고)
- 함선 비용: 건십200 · 프리깃320 · 순양함450 · 구축함520 · 배틀십800 · 배틀크루저880 SC.
- 성계 보상은 주로 **영입(카이/세라/밀라)·해금(배틀십)·희귀자원(dm)** — SC 직접 보상은 적고, 전투 승리 XP·자원 채굴(ti/dm)·정복 개발이 주 수입원.
- 7-4 조합(재료 기반 대체 경로)·유령시장(T5 exotic, 고가)이 후반 자원 소비처로 추가됨.
- **판단**: 경제는 "영입·성장" 중심이라 SC 인플레는 낮음. 플레이테스트에서 "중반 무기/함선 구매력이 따라오는가" 확인 필요.

## 4. 자동 오토배틀 실측 (2026-07-08, 사용자 요청)
전술 오토배틀(🤖 BattleScene 전체 AI 시뮬)로 실제 성계 편성을 돌려 예측을 경험 검증. 모의 전투(관제실 에디터)에서
`_summonEnemy`(7-3 헬퍼)로 성계별 편성을 재스폰하고 `node.threatLevel`을 맞춰 스케일 일치. 결과:

| 성계 | 아군 구성 | 결과 | 턴 | 비고 |
|---|---|---|---|---|
| s1 | 3척·무기 미장착(최악) | **WIN** | 3 | 무손실(3/3 생존). 튜토리얼 난이도 확인 |
| s2 | 3척·T1 레이저 | **LOSS** | 3 | 전멸 |
| s2 | 3척·T2+레벨×1.3 | **LOSS** | 3 | 장비 올려도 전멸 |
| s4 | 6척·T3+레벨×1.6 | 장기전 | 9턴 시점 워든 40%·아군 5/6 | 이후 맵 교착으로 grind-wipe(아래) |

**해석**:
- **s1 = 확정 easy** (무기 없이도 3턴 무손실 완승).
- **s2 = 실측상 가장 뚜렷한 벽**: 3척 함대는 T1/T2 무관 3턴 전멸. 리포트 §2-4(s1→s2 진입장벽)를 강하게 지지 →
  **조정 우선순위 1위 확정**. 단 아래 오토배틀 한계로 **실제 난이도는 이보다 낮다**(과대평가).
- **s4 워든 = 장기전 확인**: 6척 T3 함대가 9턴에 워든 40%까지·아군 5/6 생존 → §2-3 "8~9턴 장기전, winnable" 예측과 일치.

**오토배틀 실측의 한계(과대평가 요인) — 반드시 감안**:
1. **함대 규모**: 재스폰 하네스가 s2를 3척으로 테스트(실제 s2 진입 시 카이 합류로 4척). 4번째 함선 부재가 s2 전멸을 크게 부풀림.
2. **AI 미숙**: 오토배틀 아군 AI는 가르 **측후방 약점(7-3)**·집중사격·측면 보너스·스킬/소모품을 활용하지 못함 → 인간 플레이보다 훨씬 불리.
3. **맵 교착(신규 발견)**: s4 테스트에서 워든(이동1·사거리4)이 Twin Expanse 중앙 void 간극 너머로 카이팅되자
   오토배틀 아군이 재접근 못 하고 차원 파동(30/턴)에 갈려 9→23턴 grind-wipe. **밸런스 수치가 아니라 AI/맵 상호작용 이슈** —
   가림 지형(void 간극) 맵에서 원거리·카이팅 적 상대 시 자동전투 아군이 정지하는 패턴. **7-6/AI 폴리시에서 별도 점검 대상**
   (예: 경로 없으면 최근접 도달 가능 칸으로 전진, 또는 원거리 무기 우선 타게팅).

**실측 종합 판단**: 예측 곡선(§1~2)과 방향 일치. s2 진입 벽이 실측에서도 최우선. 단 오토배틀은 난이도를 체계적으로
과대평가하므로 **절대 승률이 아니라 상대 곡선·정성 신호로만 사용** — 정밀 조정은 여전히 7-6 인간 플레이테스트 필요.

## 5. 결론 & 다음 단계
1. **곡선 자체(개별 스케일)는 건강** — threat 단조 증가. 실측도 방향 일치.
2. 플래그 3항목(가르/워든/s1→s2)은 정량상 **처치 3~9턴 범위로 파탄 아님**. 가르·워든은 7-3 완화 장치(약점·텔레그래프) 보유.
3. **실조정 보류** — 7-6 완주 플레이테스트에서 체감(승률·전투 길이·스트레스 지점) 수집 후, 위 "만약 조정한다면" 노브를 관제실로 튜닝 → Export → 반영.
4. 조정 시 우선순위(실측 반영): **s1→s2 진입 완만화(1위 확정)** > 워든 장기전 페이스 > s2/s3 골짜기 > 가르 hp(약점으로 이미 커버, 최하).
5. **AI 폴리시 후행 항목(신규)**: void 간극 맵에서 자동전투 아군이 카이팅 적에 재접근 못 하는 정지 패턴 — 7-6 또는 별도 AI 개선 시 점검.

## 부록. 계산 스크립트
`scripts/`에 상시화하지 않고 1회성 분석으로 수행(스캐폴드는 세션 스크래치패드). 재현 시: 성계별 ΣHP/ΣATK = Σ(enemy_stats × threat 스케일), 유효피해 = atk×100/(100+armor), 보스 처치턴 = hp / Σ(함대 유효딜).
