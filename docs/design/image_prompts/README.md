## 공통 작업 절차 (모든 카테고리 동일)

1. 프롬프트 복사 → 이미지 생성 (1프롬프트 = 1장. 여러 개 한 번에 요청 금지)
2. 마음에 들 때까지 재생성 (4~8회 정상)
3. 투명배경 필요 항목(각 파일에 표기): 단색 배경으로 생성됐으면 `python scripts/remove_bg.py` 처리
4. 지정된 크기로 리사이즈 (생성은 1024×1024, 저장 시 축소)
5. `public/assets/{지정된 파일명}` 으로 저장 → 게임 재시작 시 자동 반영 (코드 수정 불필요, 폴백 자동 해제)
